# **指纹识别**



web网站指纹指的是网页中的一些关键字，特点文件的md5值，特征码，第三方插件，和一些框架独有的目录文件；指纹识别指我们通过利用正则表达式匹配这些特殊信息，识别web应用程序的名称和版本，语言类型，第三方组件以此来收集有用信息，方便后续攻击使用。
